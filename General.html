Formatos para especificar los colores

    In HTML, you can specify colors using various methods, including: 
    
    - RGB Values: You can represent colors using the RGB (Red, Green, Blue) model, where you define the amount of red, green,
    and blue in a color. For example, rgb(255, 0, 0) represents pure red.
    
    - HEX Values: Hexadecimal color codes are a popular way to specify colors. They start with a # followed by six hexadecimal
    digits, representing the color's RGB values. For instance, #FF0000 is equivalent to pure red.
    
    - HSL Values: HSL (Hue, Saturation, Lightness) is another color representation method. You define the color based on its
    hue, saturation, and lightness. For example, hsl(0, 100%, 50%) represents pure red.
    
    - RGBA Values: RGBA (Red, Green, Blue, Alpha) is an extension of RGB that includes an additional alpha channel to control
    transparency. For instance, rgba(255, 0, 0, 0.5) represents semi-transparent red.
    
    - HSLA Values: Similar to RGBA, HSLA includes an alpha channel to control transparency using the HSL model. For example,
    hsla(0, 100%, 50%, 0.5) represents semi-transparent red.
